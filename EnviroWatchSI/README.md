# EnviroWatch SI

## Opis projekta

EnviroWatch SI je projekt digitalnega dvojčka okoljskega stanja v Sloveniji. Cilj projekta je zbiranje, obdelava in prikaz podatkov o kakovosti zraka, vremenskih razmerah in vodostajih rek.

Ta modul predstavlja začetno fazo projekta, kjer se podatki pridobivajo iz spletnih virov in razčlenjujejo v uporabno obliko.

## Struktura projekta

* `model/` – podatkovni modeli
* `parser/` – razčlenjevanje podatkov (JSON, HTML, ...)
* `network/` – pridobivanje podatkov iz spletnih virov
* `Main.kt` – vstopna točka aplikacije
